# Tab Gear

> Chrome extension for tab management

### Installation

```
npm install
```

### Usage

```
npm start
```

### Build

```
npm run build
```

### Testing

```
npm test
```

### Publication

* Upload extension using [Developer Dashboard](https://chrome.google.com/webstore/developer)
